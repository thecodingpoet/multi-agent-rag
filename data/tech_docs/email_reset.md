# Email Password Reset

## Self-Service Reset

### Prerequisites
- Mobile phone or alternate email registered
- Account not locked or disabled

### Reset Steps

1. Go to https://mail.company.com → Click "Forgot Password?"
2. Enter company email → Choose verification method (SMS or email)
3. Enter verification code
4. Create new password:
   - Minimum 12 characters
   - Include uppercase, lowercase, number, special character
   - Cannot reuse last 5 passwords
5. Confirm and click "Reset Password"

## Account Locked?

- Wait 30 minutes for auto-unlock, OR
- Contact IT Help Desk: ext. 5555 or helpdesk@company.com

## Need Help?

### IT Help Desk
- **Phone**: ext. 5555 or 1-800-555-0199
- **Email**: helpdesk@company.com

### Required Info
- Full name, Employee ID, Department, Manager name
- Security verification: DOB, last 4 SSN, work location

## Password Best Practices

- Change every 90 days
- Never share with anyone
- Use unique passwords for each account
- Use a password manager
- Enable MFA

## Troubleshooting

**"Password doesn't meet requirements"**
- Check all character types included
- Verify 12+ characters
- Ensure not reusing recent password

**"Verification code not received"**
- Check spam folder
- Verify phone number on file
- Wait 2 minutes, request new code

**"Account not found"**
- Verify complete email address
- Check for typos
- Contact IT if recently changed departments
