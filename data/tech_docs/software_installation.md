# Software Installation and Requests

## Self-Service Software

### Available Without Approval

**Productivity**: Office 365, Adobe Reader, Zoom, Teams, Slack, Notepad++, 7-Zip
**Development**: VS Code, Git, Python, Node.js, Postman, Docker Desktop*
**Browsers**: Chrome, Firefox, Edge

*Docker requires manager approval

### Installation Steps

1. Open Software Center (Windows: Start Menu | Mac: Applications → "Self Service")
2. Search for application
3. Click "Install"
4. Auto-updates enabled for approved apps

## Requesting New Software

### Requires IT Approval
- Software not in Software Center
- Paid/licensed software
- Tools requiring elevated privileges
- Specialized applications

### Request Process

1. Check Software Center and approved list: https://it.company.com/approved-software
2. Submit at https://helpdesk.company.com → "Software Request"
3. Include: Software name/version, vendor, business justification, # licenses, cost

### Approval Timeline
- **Free/Open Source**: 3-7 days
- **Licensed (existing)**: 5-10 days
- **New Licensed**: 15-30 days
- **Enterprise**: 30-60 days

## Software Policies

### Prohibited
❌ P2P file sharing, unauthorized remote access, personal VPNs, crypto mining, unlicensed software

### Restricted (Requires Approval)
- Cloud storage (approved alternatives exist)
- Database management systems
- Virtual machine software
- Screen recording tools

### Developers
Broader permissions for languages, IDEs, testing tools, version control, package managers (reviewed quarterly)

## Updates

**Automatic**: OS security patches, Office, antivirus, browsers
**Manual** (IT-managed): Enterprise apps, databases, dev environments

**Schedule**: Critical within 24 hrs | Important within 7 days | Feature updates quarterly

## Uninstalling

**Self-Service**: Windows Settings → Apps | Mac Applications → Trash | Software Center
**IT-Assisted**: Enterprise software, licensed apps (email helpdesk@company.com)

## Mobile Apps

**Approved**: Office Mobile, Teams, Outlook, Authenticator, OneDrive, Adobe Mobile
**BYOD**: Install Company Portal app, work/personal data separated

## Browser Extensions

**Pre-Approved**: LastPass, Grammarly, Adobe Acrobat, Microsoft Editor
**New Requests**: 2-3 day security review, official stores only
